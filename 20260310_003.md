## GameState

```typescript
interface GameState {
  gold: number         // 보유 골드
  goldPerSec: number   // 초당 수익
  bundleCount: number  // 추가된 라인 수
}
```

---

## 골드 획득 규칙

- RE 도달 시 골드 +1 (기본값, 추후 Product 등급에 따라 변경)
- 초당수익 = 최근 1초간 RE 도달 아이템 수

---

## 라인 확장 비용

- config.ts에 설정값 관리
  - `BUNDLE_COST_BASE`: 1 (기본 비용)
  - `BUNDLE_COST_EXPONENT`: 2 (지수)
- 비용 공식: `BASE * (bundleCount + 1) ^ EXPONENT`
  - 1번째: 1 * 1^2 = 1
  - 2번째: 1 * 2^2 = 4
  - 3번째: 1 * 3^2 = 9
- 버튼에 비용 표시, 골드 부족 시 비활성화
- 구매 시 bundleCount 증가, gold 차감

---

## RS 스폰 규칙

- PR 3개가 동시에 생산 시도 → 동일 틱에 RS에 3개 스폰되는 문제 발생
- 수정: 스폰 전 RS 중심 기준 itemSize 이내에 아이템 존재하면 스킵
- RS에는 항상 최대 1개만 대기, 초과 생산은 버림
