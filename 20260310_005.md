## Factory 설계

```typescript
interface Factory {
  row: number
  col: number
  built: boolean
  type: 'WA' | 'PA' | 'PK'
  grade: number           // 0 = 미업그레이드, 1~20
  level: number           // 0 = 미업그레이드, 1+ (처리 속도)
  dir: 'UP_TO_DOWN' | 'DOWN_TO_UP'  // UP_TO_DOWN: 위에서 집어 아래로
}
```

### 타입별 등급 보너스
| 타입 | 등급당 보너스 | 최대 (20등급) |
|------|--------------|--------------|
| WA (세척) | +20% | +400% |
| PA (가공) | +30% | +600% |
| PK (포장) | +50% | +1000% |

## Item 추가 필드

```typescript
interface Item {
  // 기존 필드...
  waBonus: number    // 누적 WA 보너스
  paBonus: number    // 누적 PA 보너스
  pkBonus: number    // 누적 PK 보너스
  waGrades: number[] // 처리된 WA 등급 목록 (중복 방지)
  paGrades: number[] // 처리된 PA 등급 목록 (중복 방지)
  pkGrades: number[] // 처리된 PK 등급 목록 (중복 방지)
}
```

### 처리 규칙
- 동일 등급의 WA/PA/PK는 1번만 처리 가능
- 최대 처리: WA 20번 + PA 20번 + PK 20번

## 골드 계산

```
finalGold = value × (1 + waBonus) × (1 + paBonus) × (1 + pkBonus)

예시: WA=100%, PA=200%, PK=50%
= value × 2.0 × 3.0 × 1.5 = value × 9
```

## FA 처리 상태머신

```
IDLE → 입력 레일에 아이템 감지
     → PICKING (getFactoryPickTime(level) ms)
     → PROCESSING (보너스 적용)
     → WAITING (출력 레일 점유 시 대기)
     → PLACING (출력 레일에 아이템 내려놓기)
     → IDLE
```

- WAITING: 출력 레일이 비워질 때까지 아이템 들고 대기
- WAITING 중 입력 레일 픽업 없음

## 공장 탭 UI

- FA 셀 목록 (보드 위치 표시)
- 각 FA마다:
  - 건설 버튼 (미건설 시)
  - 타입 선택: WA / PA / PK (언제든 무료 변경)
  - 방향 선택: ↓ / ↑ (언제든 무료 변경)
  - 등급 업그레이드 버튼 + 비용
  - 레벨 업그레이드 버튼 + 비용
