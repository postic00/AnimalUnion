## 포인트 사용처

1. **아이템 가치 레벨업** — 등급별 개별 레벨업 (`itemValueLevels[]`)
2. **동물 해금** — prestige points 소비
3. **동물 업그레이드** — prestige points 소비
4. **환생 초기화** — itemValueLevels + animals 초기화, 소비 포인트 환급

## 동물 시스템

### 동물 종류 (60마리)
- 1.햄스터 ~ 20.햄스터 (`hamster1`~`hamster20`)
- 1.고양이 ~ 20.고양이 (`cat1`~`cat20`)
- 1.강아지 ~ 20.강아지 (`dog1`~`dog20`)
- 토끼 제거

### 설계 원칙
- 동물은 타입 구분 없음 — 수치(stat)만 보유
- FA에 배치 시 해당 공장 보너스에 animalStat 추가
- 공장 1개당 동물 1마리 배치 (같은 동물 여러 공장 중복 배치 가능)
- 포인트로 해금, 포인트로 stat 업그레이드
- 환생 후 동물 배치(animalId) 유지

### Animal 타입
```typescript
interface Animal {
  id: AnimalId    // 'hamster1' ~ 'dog20'
  level: number   // 업그레이드 레벨 (1+)
  unlocked: boolean
}
```

### Factory 수정
```typescript
interface Factory {
  // ...
  animalId: AnimalId | null  // 배치된 동물
}
```

### balance.ts 수식
```typescript
getAnimalUnlockCost(): number
getAnimalUpgradeCost(level): number
getAnimalStat(level): number
getItemValueResetRefund(itemValueLevels): number
getAnimalResetRefund(animals): number
```
