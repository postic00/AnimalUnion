## 2. 세이브/로드 — localStorage

### 저장 대상
- `gameState` 전체
- `board` (Cell[][] 직렬화)

### 저장 키
```typescript
SAVE_KEY = 'animal-union-save'
```

### 저장 시점
- 탭 변경 시
- 환생 시
- 일정 주기 자동저장 (30초)

### 로드 시점
- 앱 초기 마운트 시 localStorage 확인 → 있으면 복원

### 저장 포맷
```typescript
{
  version: string       // '1.0.0' — 추후 마이그레이션 대비
  savedAt: number       // Date.now()
  board: Cell[][]
  gameState: GameState
}
```
