# 동물노동조합 - 실행계획 20260312_010

## 목표
시작 로고 화면 + 로딩 화면 + 앱 아이콘 (SVG + PNG)

---

## 1. 로고 화면

### 컨셉
- 동물(🐹🐱🐶) + 마라탕후루(🍲🍡) 이모지 조합
- 배경: 주황/빨강 그라데이션 (마라탕 컬러)
- 텍스트: "동물노동조합"

### 구성
```
[배경: 주황→빨강 그라데이션]

  🐹 🐱 🐶
동물노동조합
  🍲 🍡
```

---

## 2. 로딩 화면

### 동작
- 앱 시작 시 로고 화면 표시
- 게임 상태 로드 완료 시 자동 전환
- 전환 효과: 페이드아웃 (0.5초)

### 최소 표시 시간
- 1초 (너무 빠르게 사라지지 않도록)

---

## 3. 앱 아이콘

### SVG 생성
- 1024×1024 기준
- 배경: 주황→빨강 그라데이션
- 중앙: 이모지 조합

### PNG 생성 (Node.js 스크립트)
- 패키지: `sharp`
- 출력 사이즈 (Capacitor 기준)

| 파일명 | 사이즈 |
|--------|--------|
| icon-1024.png | 1024×1024 |
| icon-512.png | 512×512 |
| icon-192.png | 192×192 |
| icon-144.png | 144×144 |
| icon-96.png | 96×96 |
| icon-72.png | 72×72 |
| icon-48.png | 48×48 |

출력 경로: `public/icons/`

---

## 수정 파일 목록

- `src/components/SplashScreen.tsx` — 로고/로딩 화면 (신규)
- `src/components/SplashScreen.module.css` — 신규
- `src/App.tsx` — SplashScreen 조건부 렌더링
- `public/icons/icon.svg` — 앱 아이콘 SVG (신규)
- `scripts/generate-icons.mjs` — PNG 생성 스크립트 (신규)
- `package.json` — sharp devDependency 추가

---

## 원칙
- 로딩 중 게임 루프 시작 안 함 (SplashScreen이 사라진 후 시작)
- 최소 표시 시간 1초 보장
- sharp는 devDependency (빌드 결과물에 미포함)
