## 6. 공장 탭 (FactoryTab)

- 타입별 카드 색상: WA(파랑) / PA(보라) / PK(오렌지)
- 레이아웃 (cardBody):
  - 좌: 타입 아이콘(32px) + 동물 배지(22px, 우하단 겹침) + 공장명 + Lv뱃지
  - 우: 1행 - 등급(‹N›) + 업그레이드버튼(90×36px)
       2행 - 타입콤보 + 방향콤보 (각 90×36px)
- ComboBox: 선택값만 표시, 클릭 시 위로 펼쳐짐
  - 선택 항목: 타입색 + ✓ 체크
- 동물 없으면 회색 ✕ 배지
- 층 헤더: 배경 없이 텍스트 + 구분선 스타일 (15px/13px)
- 레벨 뱃지: 40×20px 고정
