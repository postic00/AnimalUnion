## 1. 공장 탭 - PA(가공) 등급 제한

PA 타입은 레시피가 있는 등급(4등급~)부터만 선택 가능

### 구현
```ts
// FactoryTab.tsx
const PA_MIN_GRADE = Math.min(...Object.keys(RECIPES).map(Number))  // = 4

// ‹ 버튼 disabled 조건
disabled={factory.grade <= (factory.type === 'PA' ? PA_MIN_GRADE : 1)}
```

### 타입 변경 시 자동 보정 (App.tsx)
```ts
const handleSetFactoryType = (row, col, type) => {
  const paMinGrade = Math.min(...Object.keys(RECIPES).map(Number))
  // PA로 변경 시 현재 등급이 4 미만이면 4로 보정
  const grade = type === 'PA' && f.grade < paMinGrade ? paMinGrade : f.grade
}
```

- WA(세척), PK(포장): 1등급부터 전체 표시 (변경 없음)
- PA(가공): 4등급(레시피 최솟값)부터만 표시

## 3. 클리커 - 링 100% 시각 피드백 + 스폰 잠금

### 문제
빠른 연속 클릭 시 "링 채우기 클릭"과 "스폰 클릭"이 같이 처리되어 자동 스폰처럼 보임

### 해결: 250ms 스폰 잠금 (spawnUnlockTimeRef)
```ts
// 링이 막 100%가 될 때
if (next >= threshold) spawnUnlockTimeRef.current = now + 250

// 스폰 시도 시
if (clickCount >= threshold) {
  if (now < spawnUnlockTimeRef.current) return prev  // 잠금 중 → 무시
  // 스폰 처리
}
```

### 링 100% 시 pulse 애니메이션 (TabBar)
```ts
// isFull 판별
const isFull = clicker.clickCount >= clicker.threshold && clicker.clickCount > 0

// 클래스 적용
className={`${styles.clickerButton}${isFull ? ` ${styles.clickerButtonFull}` : ''}`}
```

```css
.clickerButtonFull {
  animation: clickerPulse 0.7s ease-in-out infinite;
}

  0%, 100% { box-shadow: 0 4px 16px rgba(49,130,246,0.45); }
  50%       { box-shadow: 0 4px 28px rgba(49,130,246,0.85), 0 0 0 6px rgba(49,130,246,0.25); }
```

## 4. 생산기 - 수량 연동 생산 주기

### 문제
수량(quantity) 레벨이 높아도 아이템이 같은 빈도로 배출 → 고수량 아이템이 너무 빠르게 나옴

### 해결
```
fullInterval = getProducerInterval(level) × quantity
```

- 링 채우는 시간 = `fullInterval`
- 링 1바퀴 완료 = 아이템 1개(quantity 포함) 배출

### 예시 (레벨 24, quantity 16)
```
getProducerInterval(24) = 10000 / (24 × 1) = 416ms
fullInterval = 416ms × 16 = 6.7초
→ 6.7초마다 아이템 1개(×16) 배출
```

### 이전 방식(카운터 누적) 제거
카운터 방식은 링이 빠르게 채워졌다가 16번째에만 아이템이 나와 시각적으로 혼란스러움
→ `produceCountersRef` 완전 제거, `fullInterval` 단일 타이머로 통합

### 수정 파일
- `src/hooks/useGameLoop.ts`
  - `produceCountersRef` 제거
  - 생산 조건: `timer >= fullInterval`
  - 프로그레스: `timer / fullInterval`
