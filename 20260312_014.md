## 작업 내용

### `public/fa-cell-preview.html` 개선

#### 레이아웃 구조 변경 (z-index 레이어링)
- `hand-wrap`을 `.cell` 내부에서 `.factory-col`의 첫 번째 자식으로 이동
- z-index 계층: `rail(3)` < `hand-wrap(5)` < `cell(7)`
- 효과: 셀 배경이 팔(arm) 부분을 가려주고, 발(paw)만 레일 영역에서 보임

#### GRABBING 애니메이션
- 위 레일 아이템: 손이 닿자마자 즉시 사라짐 (손 아래 숨겨진 느낌)
- 손에는 아이템 표시 없음 (손 아래 숨겨진 상태)
- IDLE 복귀 시 위 레일 아이템 부드럽게 다시 나타남

#### PLACING 애니메이션 (신규)
- SVG 180도 회전 → 발끝이 아래 레일 방향을 향함
- 손이 아래 레일에 닿는 순간(200ms) 아래 레일 아이템이 나타남
- 손에는 아이템 표시 없음 (이미 내려놓은 상태)
- 손만 빈 채로 천천히 올라와 사라짐

#### 애니메이션 타이밍
- GRABBING: 2.0s ease-out (10%에 레일 도달, 나머지 천천히 복귀)
- PLACING: 2.0s ease-out (10%에 아래 레일 도달, 나머지 천천히 복귀)
- PROCESSING resting 위치: translateY(5px) — 셀 하단 경계 초과 방지
