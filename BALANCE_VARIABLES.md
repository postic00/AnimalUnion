# 밸런스 변수 구조

## 네이밍 규칙

- 각 그룹은 `{그룹}_{PROC|COST}_{BASE|RATE|EXP|ACC}` 형태로 구성
- **PROC**: 효과 관련 변수
- **COST**: 업그레이드 비용 관련 변수
- **BASE**: 기본값
- **RATE**: 배율 (곱셈/나눗셈)
- **EXP**: 지수
- **ACC**: 가속 계수 (레벨 제곱에 비례한 가속)

## 공식

| | 공식 |
|-|------|
| **COST** | `BASE * RATE^(lv * EXP + lv² * ACC)` |
| **PROC** | `BASE * lv * EXP * ACC` |

## 그룹 목록

| 그룹 | 설명 |
|------|------|
| `PR` | 생산기 생산 주기 |
| `FA` | 공장 처리 시간 |
| `IT` | 아이템 가치 |
| `IC` | 아이템 수량 |
| `BF` | 버퍼 용량 |
| `RL` | 레일 속도 |
| `AM` | 동물 스탯 |
| `BD` | 라인 추가 |
