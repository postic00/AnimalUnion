# 동물노동조합 - 게임 설계 문서

## 진행 현황

### 완료
- 보드 레이아웃 + 레일 이동
- 생산자(PR) 건설/업그레이드
- 공장(FA) 건설/타입/등급/레벨/방향
- 클리커
- 라인 확장
- 환생 시스템 (조건/포인트/리셋/초기화)
- 동물 시스템 (60마리 해금/업그레이드, 공장 배치, 환생 후 배치 유지)
- 숫자 포맷 (4자리, k/m/b/t/aa...)

### 미완료 — 빠르게 가능 (코딩 위주)
- Capacitor + AdMob (AAB 빌드 포함)
- 세이브/로드
- 아이템 탭 (등급별 가치 연결)
- 동물 배치 UI (해금된 동물 선택)
- 설정
- 광고 기능 (게임 속도 2배, 가격 올리기) — 수치는 config.ts
- 리더보드 + 시즌 (Firebase)
- 튜토리얼
- 시작 로고 + 로딩 화면
- 밸런스 조절
- 크레인(CR) 시스템

### 미완료 — 시간 필요 (리소스/외부 의존)
- 전체 디자인/UI 정리
- BGM + 효과음
- 앱 호스팅 (Vercel/Netlify 등 정적 호스팅)

## Field 정의

- 구조: **7 * x 그리드** (7열 고정, 골드로 행 추가)
- 각 셀에 블록을 배치하여 아이템 흐름 구성

### 셀 블록 타입

| 약어 | 이름 | 설명 |
|------|------|------|
| RS | Rail Start | 레일 시작점 — 아이템 흐름 시작 |
| RE | Rail End | 레일 종료점 — 아이템 판매/수집 |
| RL | Rail Left | 레일 좌 — 아이템 왼쪽으로 이동 |
| RR | Rail Right | 레일 우 — 아이템 오른쪽으로 이동 |
| RU | Rail Up | 레일 위 — 아이템 위로 이동 |
| RD | Rail Down | 레일 아래 — 아이템 아래로 이동 |
| PR | Production | 생산 — 파 자동 생산 후 레일로 배출 |
| FA | Factory | 공장 — 배치 후 타입 선택 (WA/PA/PK) |
| CR | Crane | 크레인 — 아이템을 들어올려 다른 레일로 이동 |
| EM | Empty | 빈블록 — 비어있는 셀 |

## 도메인 객체

| 객체 | 설명 | 골드 레벨 | 환생 레벨 |
|------|------|-----------|-----------|
| Product | 원재료/제품 (Lv.01~20) | | ✅ 등급 해금 |
| Field | 7*x 그리드 (수확/생산) | ✅ 행 추가 | ✅ 수확량/속도 강화 |
| Factory (FA) | 공장 슬롯 — 배치 후 타입 선택 | ✅ 레벨업 | ✅ 가공 속도 강화 |
| ↳ WA (Wash) | 세척 공장 | | ✅ 세척 능력 강화 |
| ↳ PA (Process) | 가공 공장 | | ✅ 가공 능력 강화 |
| ↳ PK (Pack) | 포장 공장 | | ✅ 포장 능력 강화 |
| Animal | 공장에 배치되는 동물 | | |
| ↳ WashAnimal | 세척 동물 | | ✅ 세척 능력 강화 |
| ↳ ProcessAnimal | 가공 동물 | | ✅ 가공 능력 강화 |
| ↳ PackAnimal | 포장 동물 | | ✅ 포장 능력 강화 |
| Crane (CR) | 아이템 재투입 (양방향) | ✅ 해금/강화 | |
| Leaderboard | 글로벌 순위 | | |

### Factory 속성

```
Factory (FA)
├── type: WA | PA | PK   ← 배치 후 선택
├── level: number        ← 골드로 업그레이드
└── animal: Animal       ← 배치된 동물 (능력 강화)
```
